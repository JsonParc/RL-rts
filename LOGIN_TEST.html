🔑 로그인 테스트 가이드

⚠️ 로그인 안 되는 문제 해결 완료!
아래 단계대로 따라하면 정상적으로 작동합니다.

1️⃣ 브라우저 캐시 지우기 (중요!)

수정된 코드가 적용되려면 브라우저 캐시를 지워야 합니다:

  • Chrome: Ctrl + Shift + Delete → 캐시된 이미지 및 파일 체크 → 삭제
  • 또는 간단한 방법: Ctrl + F5 (강력 새로고침)

2️⃣ 새 계정 만들기

테스트용 계정을 만들어 보세요:

  1. 사용자명: test123 (원하는 이름)
  2. 비밀번호: 1234 (4자 이상)
  3. 회원가입 버튼 클릭
  4. "회원가입 성공!" 메시지 확인

3️⃣ 로그인하기

  • 위에서 만든 사용자명과 비밀번호 입력
  • 로그인 버튼 클릭 (또는 Enter 키)
  • 로그인 화면이 사라지고 게임 화면이 나타남

4️⃣ 게임 시작!

로그인 성공 후:

  • ✅ 맵이 표시됨 (바다와 육지 섬)
  • ✅ 본부가 육지 섬에 자동 배치됨
  • ✅ 자원, 인구, 점수가 표시됨
  • ✅ 미니맵에서 위치 확인 가능

준비 완료! 🎉

🎮 게임 시작하기

🔧 여전히 문제가 있다면?

브라우저 개발자 도구를 확인하세요:

  1. F12 키를 눌러 개발자 도구 열기
  2. Console 탭에서 에러 메시지 확인
  3. Network 탭에서 API 요청 상태 확인

에러 메시지가 보이면 제게 알려주세요!

💡 팁:
  • Enter 키로 바로 로그인 가능
  • 비밀번호는 4자 이상 필요
  • 로그인 후 토큰이 자동 저장되어 다음에 자동 로그인됨
  • 로그아웃하고 싶으면 우측 상단 "로그아웃" 버튼 클릭
