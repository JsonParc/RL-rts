✅ 서버 상태: 확인 중...

로컬 서버가 실행 중이면 아래 버튼으로 바로 접속할 수 있습니다!

🎮 게임 시작하기 🌐 배포 가이드

1️⃣ 로컬에서 플레이하기 (현재)

서버가 이미 실행 중입니다! 위의 "게임 시작하기" 버튼을 클릭하세요.

  • 회원가입 후 로그인
  • 자동으로 육지 섬에 본부 배치됨
  • 건물 건설 → 군함 생산 → 적 공격!

2️⃣ 온라인 배포하기

친구들과 함께 플레이하려면 게임을 온라인에 배포해야 합니다.

추천 플랫폼:

  • Render.com - 무료, 안정적, Socket.io 완벽 지원 ⭐
  • Railway.app - 매우 빠름, GitHub 자동 연동
  • Glitch.com - 가장 쉬움, 웹에서 바로 편집

상세한 방법은 배포 가이드를 참고하세요.

3️⃣ 게임 비율 정보

mapandunit.png 파일의 비율을 그대로 반영했습니다:

요소 크기 비율
🏗️ 건물 (초록색 사각형) 100 x 100 100% (기준)
💎 자원 (빨간색 선) 반지름 60 60%
🚢 군함 (파란색 선) 30 x 30 30%

4️⃣ 조작법

  • 좌클릭 드래그: 유닛 선택
  • 우클릭: 이동/공격 명령
  • W/A/S/D: 카메라 이동
  • 마우스 휠: 줌 인/아웃
  • 미니맵 클릭: 카메라 순간 이동

⚠️ 서버가 닫히면?

VSCode 터미널에서 Ctrl + C를 눌러 서버를 종료하면 게임도 종료됩니다.

다시 시작하려면: npm start

24시간 온라인 유지: 위의 배포 가이드를 참고하여 온라인 서버에 배포하세요!
