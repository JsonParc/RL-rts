# ⚓ Naval Warfare RTS - 해상 전투 실시간 전략 게임

대규모 멀티플레이어 온라인 해상 전투 RTS 게임입니다. 스타크래프트 스타일의 게임플레이로 군함을 생산하고 적과 싸우며 해양 패권을 차지하세요!

## 🎮 게임 특징

### 맵 & 환경
- **4000x4000** 크기의 대규모 해상 맵
- 바다가 주 지형이며, 전략적 위치에 육지 섬 배치
- 육지에서만 건물 건설 및 자원 채취 가능
- 적대적 NPC 함선이 바다를 순찰

### 유닛 시스템 (군함)
- **구축함** (Destroyer): 빠르고 저렴한 경량 전투함
    - 비용: 150 자원, 인구: 2
    - HP: 300, 공격력: 25
- **순양함** (Cruiser): 균형잡힌 중형 전투함
    - 비용: 300 자원, 인구: 3
    - HP: 500, 공격력: 40
- **전함** (Battleship): 강력한 중장갑 전투함
    - 비용: 600 자원, 인구: 5
    - HP: 1000, 공격력: 80
- **항공모함** (Carrier): 최강의 함대 기함
    - 비용: 800 자원, 인구: 6
    - HP: 800, 공격력: 60

### 건물 시스템
- **본부** (Headquarters): 시작 기지
    - HP: 1500
- **조선소** (Shipyard): 군함 생산 시설
    - 비용: 200 자원, HP: 800
- **자원 부두** (Resource Dock): 자원 채취 시설
    - 비용: 150 자원, HP: 600
- **방어 타워** (Defense Tower): 방어 구조물
    - 비용: 250 자원, HP: 700
- **해군 사관학교** (Naval Academy): 업그레이드 시설
    - 비용: 300 자원, HP: 700

### 멀티플레이어 기능
- **실시간 온라인 대전**: 여러 플레이어가 동시에 같은 맵에서 플레이
- **로그인/계정 시스템**: 개인 계정으로 진행 상황 저장
- **오프라인 공격**: 로그아웃 상태에서도 다른 플레이어가 공격 가능하며 피해 반영
- **실시간 랭킹 시스템**: 상위 10위 실시간 표시
    - 점수 = 자원 + (인구 × 100) + (전투력 × 50)

## 🎯 크기 비율 (mapandunit.png 기준)

게임 내 모든 요소의 크기는 서로 비율을 맞춰 설계되었습니다:

| 요소 | 크기 | 비율 | 설명 |
|------|------|------|------|
| **건물** | 100 x 100 | 100% (기준) | 본부, 조선소, 방어타워 등 |
| **자원 구역** | 반지름 60 | 60% | 금, 자원 채취 지점 |
| **유닛 (군함)** | 30 x 30 | 30% | 구축함, 전함, 항모 등 |

이 비율은 mapandunit.png의 초록색 사각형(건물), 빨간색 선(자원), 파란색 선(유닛)의 비율을 그대로 반영합니다.

## 🚀 설치 및 실행

### 필수 요구사항
- Node.js 16.0 이상
- npm 또는 yarn

### 설치
```bash
# 의존성 설치
npm install
```

### 실행
```bash
# 서버 시작
npm start

# 또는 개발 모드 (자동 재시작)
npm run dev
```

서버가 시작되면 브라우저에서 `http://localhost:3000` 접속

### 배포 (온라인 공개)
```bash
# 배포 가이드 보기
DEPLOYMENT.html 파일 열기

# 추천 플랫폼:
# 1. Render.com (가장 추천)
# 2. Railway.app (빠름)
# 3. Heroku (유명함)
# 4. Glitch.com (가장 쉬움)
```

상세한 배포 방법은 [DEPLOYMENT.html](DEPLOYMENT.html) 파일을 참고하세요.

## 🎮 조작법

### 마우스
- **좌클릭 드래그**: 유닛 선택 (박스 선택)
- **우클릭**: 선택된 유닛 이동/공격 명령
- **마우스 휠**: 줌 인/아웃
- **중간 버튼 드래그**: 카메라 이동

### 키보드
- **W/A/S/D**: 카메라 이동
- **Esc**: 건설 모드 취소

### UI
- **건물 건설 메뉴**: 화면 우측 하단
- **함선 생산 메뉴**: 조선소/본부 선택 시 표시
- **미니맵**: 화면 좌측 하단 (클릭으로 카메라 이동)
- **랭킹 패널**: 화면 우측 상단

## 🔧 기술 스택

### 백엔드
- **Node.js** + **Express**: 웹 서버
- **Socket.io**: 실시간 양방향 통신
- **better-sqlite3**: SQLite 데이터베이스
- **bcrypt**: 비밀번호 암호화
- **jsonwebtoken**: JWT 인증

### 프론트엔드
- **HTML5 Canvas**: 게임 렌더링
- **Vanilla JavaScript**: 게임 로직
- **Socket.io Client**: 실시간 통신

## 📊 데이터베이스 스키마

### users 테이블
- 사용자 계정 정보 (username, password)

### player_data 테이블
- 플레이어 게임 데이터 (자원, 인구, 점수, 기지 위치 등)

### units 테이블
- 모든 유닛 정보 (위치, HP, 타입 등)

### buildings 테이블
- 모든 건물 정보 (위치, HP, 건설 진행도 등)

## 🎯 게임 플레이 팁

1. **초반 전략**: 자원 부두를 먼저 건설하여 자원 수입 증가
2. **함대 구성**: 다양한 함선을 조합하여 균형잡힌 함대 편성
3. **방어**: 방어 타워로 기지 주변을 보호
4. **확장**: 여러 섬에 전초기지를 건설하여 영토 확장
5. **오프라인 방어**: 로그아웃 전 방어 구조물과 함대 배치 필수
